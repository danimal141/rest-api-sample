## Getting Started

### How to Develop

```
$ go get github.com/danimal141/rest-api-sample
$ cd $GOPATH/src/github.com/danimal141/rest-api-sample
$ dep init
$ dep ensure -update
$ go run main.go
```

### Test

```
$ go test ./api/v1/...
```

### Generate API documentation

```
$ npm install
$ gulp
```
